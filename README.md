# My GitHub Pages Website

A simple, clean HTML website hosted on GitHub Pages.

## Features

- 🚀 Fast and lightweight
- 📱 Fully responsive
- ✨ Easy to customize
- 🎨 Modern design

## How to use

1. Edit `index.html` to customize your content
2. Push changes to the `main` branch
3. Visit `https://owenb135.github.io` to see your site live

## Customization

- Change the title, headings, and text in the HTML
- Modify colors in the `<style>` section
- Add more sections as needed
- Link to external resources or other pages

## Need help?

Check out the [GitHub Pages documentation](https://docs.github.com/en/pages) for more information.
